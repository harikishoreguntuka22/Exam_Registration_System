# 🎓 Exam Registration System

A modern web-based Exam Registration System that allows students to register for exams, view available exams, and manage their registrations through a simple and user-friendly interface.

---

## 🚀 Features

- Student signup and login system  
- View list of available exams  
- Register for exams easily  
- View registered exams  
- Clean and responsive UI  
- Form validation and error handling  

---

## 🛠️ Tech Stack

- React + TypeScript  
- Vite  
- Tailwind CSS  
- ShadCN UI  
- Node.js (environment)  
- npm / bun  

---

## 📁 Project Structure


src/
├── components/ UI components (Navbar, Footer, etc.)
├── pages/ Application pages (Login, Signup, Exams, etc.)
├── hooks/ Custom React hooks
├── lib/ Utility functions
├── App.tsx Main application file
└── main.tsx Entry point


---

## ⚙️ Installation & Setup

### 1. Clone the repository

git clone https://github.com/harikishoreguntuka22/Exam_Registration_System.git


### 2. Navigate to project folder

cd Exam_Registration_System


### 3. Install dependencies

npm install


### 4. Run the project

npm run dev


### 5. Open in browser

http://localhost:5173
